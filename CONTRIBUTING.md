# Contributing

Thank you for your interest in contributing to **json_presenter**.

Contributions of all sizes are welcome, including:

* Bug reports
* Feature requests
* Documentation improvements
* Code contributions
* Theme improvements
* New presentation ideas

## Development Setup

Clone the repository:

```bash
git clone https://github.com/ZunKhet/json_presenter.git
```

Install the project:

```bash
pip install -e .
```

Run the test suite:

```bash
pytest
```

## Code Style

Before submitting a pull request, please ensure that:

```bash
ruff check .
black .
pytest
```

all pass successfully.

## Pull Requests

Please keep pull requests focused on a single topic whenever possible.

Include tests for new functionality.

## Questions

If you have ideas or questions, feel free to open a discussion or issue.
